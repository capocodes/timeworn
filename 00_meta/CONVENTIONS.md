# Doc conventions

- **Prefixes:** Folders use numeric prefixes (`01_vision`, …) for stable sort order.
- **One job per file:** Prefer splitting when a file covers two unrelated decisions.
- **Canon vs notes:** `01_`–`07_` = design/business truth we endorse. `00_meta/assistant/` = working memory until promoted.
- **Naming:** `level_<slug>.md`, `character_<name>.md`, `system_<name>.md`, `ep_NNN_title.md` for devlog episodes.
